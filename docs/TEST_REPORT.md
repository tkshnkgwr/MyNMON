### 2.1 テストケース詳細
- `tests::test_get_ascii_bar`:
  - 負荷 `0.0%` の時にスペースのみのバー `[          ]` が生成されること。
  - 負荷 `50.0%` の時に半分進んだバー `[====>     ]` が生成されること。
  - 負荷 `100.0%` の時に最大まで進んだバー `[=========>]` が生成されること。
  - 上記のアサーションが全て正常に満たされることを検証。

## 4. 総合評価

### 判定：**合格 (Excellent)**

sysinfo v0.30 の新しいAPI仕様への移行に続き、コマンドラインからのバージョン情報取得、ヘルプ表示オプション、エラーハンドリングを無事に追加しました。
バージョン情報は `Cargo.toml` からビルド時に動的に埋め込まれるようになり、コード内のハードコーディングの排除に成功しています。
また、外部引数パースライブラリ（`clap` 等）を使わない方針により、バイナリサイズおよびメモリ負荷への悪影響を完全に防ぎつつ、必要なオプション機能を要件通りに実現していることを確認しました。
